# Ion Trap Computational Modelization

This repository contains MATLAB implementations for simulating ion confinement in a **Paul ion trap** using numerical methods (Method of Moments, potential calculations, trajectory prediction).  
The project explores quadrupole potentials, charge distributions, and optimal trap parameters based on ion properties and electrode geometry.
